# 💰 支出管理アプリ

シンプルで使いやすい支出管理アプリケーションです。ブラウザのlocalStorageを使用してデータを保存します。

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 支出を記録する
3. カテゴリ別の集計を確認する

## ✨ 機能

- **支出記録**: 日付、カテゴリ、金額、メモを入力して記録
- **支出一覧**: 記録した支出を一覧表示、削除可能
- **カテゴリ別集計**: 7つのカテゴリで自動集計
- **フィルタ機能**: 今月/すべての支出を切り替え表示

## 📱 対応カテゴリ

- 🍽️ 食費
- 🚃 交通費
- 🎮 娯楽費
- 🛒 日用品
- 💊 医療費
- 💡 光熱費
- 📝 その他

## 💾 データ保存

データはブラウザのlocalStorageに保存されるため、ページを閉じても記録が残ります。

## 🌐 GitHub Pages での公開

このアプリをGitHub Pagesで公開する場合:

1. リポジトリの Settings → Pages に移動
2. Source を「Deploy from a branch」に設定
3. Branch を「main」、フォルダを「/ (root)」または「/expense-tracker」に設定
4. Save をクリック

数分後、`https://[ユーザー名].github.io/[リポジトリ名]/expense-tracker/` でアクセスできます。
